Lession 3: Advanced Features of HTML

Objective: Explore forms, multimedia, and responsive design.

  1. HTML Forms

    • Forms are used to collect user input.
    • Common Input Types: text, email, password, radio, checkbox, submit.

  2. Adding Multimedia

    • Video:

      Code: <video controls> <source src="video.mp4" type="video/mp4"> Your browser does not support the video tag. </video>

    • Audio:

      Code: <audio controls> <source src="audio.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio>

  3. Embedding YouTube Videos

  4. Responsive Design

    • Use the meta viewport for responsiveness: <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • Add responsive images: <img src="image.jpg" style="max-width: 100%; height: auto;">
