Structure of HTML Code:

About the HTML Code:

  1. <!DOCTYPE html>
    • Declares the document as an HTML5 document.
    • Ensures the browser interprets the HTML correctly as per modern standards.
  2. <html lang="en"></html>
    • The tag represents the root element of the webpage.
    • The lang="en" attribute specifies the language of the content as English. This helps with accessibility and SEO.
  3. <head>
    • Contains metadata about the document (not visible on the webpage).
    • Includes information such as the character set, viewport settings, page title, and links to styles or scripts.
  4. <meta charset="UTF-8">
    • Specifies the character encoding as UTF-8.
    • Ensures the webpage supports a wide range of characters, including symbols and non-English alphabets.
  5. <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • Ensures the page is responsive and displays correctly on different devices (like mobiles and desktops).
    • width=device-width: Makes the page width match the screen width.
    • initial-scale=1.0: Sets the initial zoom level.
  6. <title>
    • Sets the title of the webpage, which appears on the browser tab.
    • Example: Full-Screen Image.
  7. <style>
    • Used to define internal CSS styles for the page.
    • This block contains styling for the full-screen image.
  8. <body>
    • The <body> tag contains all the visible content of the webpage.
    • Elements like headings, paragraphs, images, videos, and more go inside this section.
  9. <img src="your-image.jpg" alt="Full Screen Image" class="fullscreen-image">
    • Displays the image on the page.
    • src="your-image.jpg": Specifies the path to the image file.
    • alt="Full Screen Image": Describes the image for accessibility (e.g., screen readers) and is displayed if the image cannot load.
    • class="fullscreen-image": Associates the image with the CSS class that styles it.
