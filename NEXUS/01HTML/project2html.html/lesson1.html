Lession 1:Introduction to HTML

Objective: Learn the basics of HTML structure and common elements.

  1. What is HTML?

    • HTML stands for HyperText Markup Language.
    • It is the standard language used to create webpages.
    • HTML uses tags to structure content.

  2. Structure of an HTML Document

    • <!DOCTYPE html>: Declares HTML5.
    • <html>: The root element.
    • <head>: Contains metadata (e.g., title, styles).
    • <body>: Contains all the visible content.

  3. Common HTML Tags

    • Headings: <h1> to <h6> (e.g., <h1> is the largest).
    • Paragraph: <p>.
    • Line Break: <br>.
    • Bold/Italic Text: <b> (bold), <i> (italic).

  4. Adding Links and Images

    • Link:

      <a href="https://www.example.com">Visit Example</a>

    • Image:

      <img src="image.jpg" alt="Description of the image">
