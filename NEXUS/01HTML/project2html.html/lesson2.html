Lession 2: HTML Page Structure and Styling

Objective: Learn the basics of HTML structure and common elements.

  1. Semantic HTML

    • Semantic tags describe the meaning of content:
    • <header>: Defines the header of the page.
    • <nav>: Navigation menu.
    • <main>: Main content.
    • <section>: A thematic grouping of content.
    • <footer>: Footer of the page.

  2. Using Div and Span

    • <div>: Groups elements together.
    • <span>: Inline container for styling.

  3. Adding Inline CSS

    • Style elements directly using the style attribute.

    Example: <p style="color: blue; font-size: 18px;">Styled Paragraph

  4. External CSS

    • Link an external stylesheet:

    Example: <head> <link rel="stylesheet" href="styles.css"> </head>
